# BOARD-T

### リンク：**https://board-laravel.herokuapp.com/category**

## 概要
データベース利用の簡易掲示板です。開発はMAMP（7.4.12）、MySQLはver5.7です。

## 基本機能
・ユーザーの登録/ログイン/ログアウト機能

・掲示板への投稿機能/投稿へのコメント機能

・マイページで自己紹介文やプロフィール設定
